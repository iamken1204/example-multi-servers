# Example of golang multiple servers
How to use:   
```
$ go get github.com/iamken1204/example-multi-servers
$ cd $GOPATH/src/github.com/iamken1204/example-multi-servers
$ go run main.go
```
visit   
* localhost:8001
* localhost:8001/foo
* localhost:8001/bar
* localhost:8002
* localhost:8002/foo
* localhost:8002/bar
